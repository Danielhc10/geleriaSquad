BARRA DE MENúS (1)

En esta barra consta de las siguientes secciones, descritas a continuación 

  • Inicio 
  • Tags

De forma predeterminada cuando accedemos a la pagina nos posiciona en la opción de inicio la cual consta de todos los menús ya antes mencionados, esta barra es independiente de que sección se encuentre activa. 

Tags

Los tags es un espacio en el cual se almacenan un un conjunto de imágenes de acuerdo con grupo de categorias o áreas donde pertenece dicha imagen, ya sea de acuerdo con su tipo de contenido o clasificación. Siendo la propia página web la encargada de la creación de dichas categorias, para posteriormente ser usadas por el usuario. 

Inicio 

La página de inicio se centra en la proyección de las diferentes estructuras que consta la pÁgina, pero su principal objetivo es mostrar las imágenes que se han estado subiendo a la pagina como una vista previa de lo que esta contiene, sin la necesidad de entrar a inicio de sesión o de registrarse. 

AREA DE REGISTRO (2)

Esta ventana de registro debe ser llenada o contestada por datos que son esenciales para la creación de un nuevo usuario, esta consta de 4 campos los cuales son:

Parametros para registro

  • Nombre de usuario (1)
  • Correo (2)
  • Contraseña (3)
  • Botón de registrar (4)

Nombre de usuario

El nombre de usuario es un dato de identificación, el cual podría ser desde el nombre de la persona o apodo, este nombre ya es de acuerdo al gusto de cada usuario. 

Correo

El correo electrónico es con el cual se iniciaría un inicio de sesión, este correo puede ser tanto Gmail Gmail, Hotmail entre otros. Siendo un requisito obligatorio para la creación de una nueva cuenta de usuario.

Contraseña

La contraseña no cumple con una medida estricta de como de ser colocada, teniendo como libertad libertad para los usuarios el uso de numeros, letras mayúsculas y signos de puntuación en el orden que prefieran o si es el caso, no hacer uso de algunos de estos. 

Botón de registrar

Por último, en esta ventana, nos encontramos con el botón de registrar el cual una vez concluido el registro nos permitirá guardar en la base de datos el nuevo usuario ya creado.

Parametros para inicio de sesión:

  • Correo
  • Contraseña
  • Botón de inicio

Nombre 

En esta opción es importante acordarse del nombre ya utilizado para la creación de un nuevo usuario (Área de registro). Sirviéndonos para poder acceder a nuestra cuenta.

Contraseña 

La contraseña, como tambien el correo es importante para el inicio de sesión seguro, es por eso por lo que ambos datos deben de ser recordados por parte del usuario. 

Botón de inicio

Al terminar con el ingreso de ambos datos anteriormente mencionados, damos por concluido nuestro inicio de sesión al posicionarnos en esta área y dar clic, de esta manera accederemos a nuestra cuenta teniendo permisos para subir imágenes, entre otras opciones.
